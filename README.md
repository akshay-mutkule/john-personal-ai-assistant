# 📖 Overview

**John – Personal AI Assistant** is a Python-based virtual assistant designed to simplify everyday computer tasks through voice interaction and automation. It understands user commands, performs various operations, and provides quick responses, making it a smart companion for improving productivity.

The project demonstrates the integration of Artificial Intelligence, speech processing, automation, and Python programming concepts into a single desktop application.
